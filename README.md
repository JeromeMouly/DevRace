# DevRace #

Site virine avec inscription des team en vue du hackathon.

### Lecture Dossier ###

#### Linux ####

**Browser Sync**

Installation :

> npm install -g browser-sync


Lancement :

> browser-sync start --server --files "*.html, css/*.css, js/*.js, pages/*.html"


### HWindows ###

**Visual Studio Code**

> Installer l'extension 

> vscode-refresh-html

ou

> Open HTML in Default Browser
